# 题库管理器（初版）

直接在浏览器打开 `index.html` 即可使用。
答题页请打开 `quiz.html`。

## 已实现功能
- 新建题库（清空当前内容）
- 新增/删除题目
- 支持题型：单选、多选、填空、简答、判断
- 编辑题干与选项（富文本区域）
- 题干和选项支持插入图片：
  - 直接粘贴截图/图片
  - 选择本地图片文件插入
- 绑定一个 JSON 文件并自动同步保存
- 导出题库为 JSON 文件
- 导入 JSON 题库文件
- 独立答题页：逐题提交判分，完成后显示成绩
- 答题页支持从仓库直接选择题库 JSON（不必本地上传）

## 数据说明
- 图片会以内嵌 `base64`（data URL）形式保存在题库 JSON 中。
- 如果图片很多，导出的 JSON 文件体积会明显变大。
- 单选/多选的正确答案保存在各个选项的 `isCorrect` 字段中（不是 A/B/C/D 文本）。

## 使用提示
- 先点“绑定同步文件”，选择要维护的 `.json` 文件。
- 后续编辑会自动同步写入该文件，也可以点“立即同步到文件”手动触发。
- 文件同步依赖 File System Access API，建议使用 Edge/Chrome 并通过 http(s) 打开页面。
- `quiz.html` 会读取 `banks.json` 作为仓库题库列表；把新的 JSON 文件加入仓库后，记得同步更新 `banks.json`。
- 答题页开始后会随机题目顺序，并对单选/多选题随机选项顺序。
